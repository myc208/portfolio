Student Grades Database in C

Console-based application for managing academic records using file I/O and structured programming in C.

Objective

To develop a lightweight, command-line application that reads, writes, and manages student records using a structured `.txt` file as a basic database.

Tools & Language

  • C Programming Language
  • Visual Studio Code
  • Standard Library I/O

Functional Breakdown

  1. File read on open (creates default if not found)
  2. Search by ID or name
  3. Insert, update, and delete student records
  4. Save back to the `.txt` file

Example Code – Open Database File


void openDb(const char *filename) {
    FILE *file = fopen(filename, "a+");
    if (file == NULL) {
        printf("Error opening file.\n");
        return;
    }

    fseek(file, 0, SEEK_END);
    if (ftell(file) == 0) {
        fprintf(file, "ID\tName\t\tProgramme\t\tMark\n");
    }
    rewind(file);
    // Parse student records...
    fclose(file);
}

Key Logic Notes

  • Uses `a+` mode to auto-create missing files
  • Validates each line before parsing
  • Dynamic memory for linked list records
  • Backed by structured design: `struct`, `Node`, and `Linked List`
