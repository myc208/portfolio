Grades Record System (C Language)

Manage student records with file persistence and console-based commands.

123456 

void saveGrades(Student *head) {
    FILE *fp = fopen("grades.txt", "w");
    while (head) {
        fprintf(fp, "%s %d\\n", head->name, head->score);
        head = head->next;
    }
    fclose(fp);
}

Features:

  • Dynamic linked list of student structs
  • Save/load from text file with validation
  • Sort, search, and update records via command input
