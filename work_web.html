Medium-like Blog Application

A group-developed blog platform inspired by Medium, featuring post CRUD operations, responsive search functionality, and AI translation attempts, hosted with Azure database.

Project Description

This group project created a blog website modeled after Medium. Tasks were divided based on team proficiency, with my focus on implementing CRUD operations for posts (create, read, update, delete) and enhancing the search bar's aesthetics and functionality in the logged-in home page. The application includes features like global user post views, attempted OpenAI integration for language translation, dynamic display sizes for responsiveness, and adherence to PHP_CodeSniffer (PHPCS) standards for code quality. The database is hosted on Google Azure for a server-client effect, enabling real-world deployment. From the report, the system architecture emphasizes secure user authentication, post management, and admin controls, with AI assisting in code generation for translation features.

Search Bar Logic (main.js - Snippet)

function attachSearchEventListeners() {
    const searchInput = document.getElementById("search-input");
    if (searchInput) {
        searchInput.addEventListener("input", function(event) {
            event.preventDefault();
            updatePosts(this.value);
        });
    }
}

function updatePosts(query) {
    const postsContainer = document.getElementById("posts-container");
    if (!postsContainer) {
        console.error("Posts container not found!");
        return;
    }

    postsContainer.innerHTML = "
Loading...
";
    postsContainer.style.display = "none";

    if (query.trim() === "") {
        window.location.href = window.location.pathname;
        return;
    }

    let xhr = new XMLHttpRequest();
    xhr.open('GET', 'home_loggedin.php?query=' + encodeURIComponent(query) + '&ajax=true', true);
    xhr.onload = function() {
        if (xhr.status === 200) {
            const tempDiv = document.createElement('div');
            tempDiv.innerHTML = xhr.responseText;
            postsContainer.innerHTML = '';
            Array.from(tempDiv.children).forEach(child => {
                postsContainer.appendChild(child);
            });
            postsContainer.style.marginTop = '0';
            postsContainer.style.paddingTop = '0';
        }
    };
    xhr.send();
}
