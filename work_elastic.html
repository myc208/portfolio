Establishing ElasticSearch DB Ecosystem

Test viability, efficiency and effectiveness of ElasticSearch DB over MSSQL for speed and functionality

Project Description

As a database engineer intern, my task was to establish a working database on elasticsearch and learn about the ecosystem that revolves around it, known as the elasticstack. Through that, I can simplify the implementation process of various other features such as real-time data logging, data visualizations, as well as the community discussion group to query and seek professional help from.

Technologies Used

ElasticSearch
Docker
Kibana
Logstash

Key Tasks

Database Implementation

  • Research and learn about ElasticSearch
  • Implementation on Docker container
  • Status monitoring via Kibana

Algorithm Development

  • Implement sample words to be flagged
  • Develop flagging algorithm
  • Testing and UAT process

Project Outcomes

Performance Improvement

ElasticSearch showed 3x faster query performance compared to MSSQL for text search operations

Team Collaboration

Daily standup meetings and knowledge sharing improved implementation efficiency
