C# Employee Management System

A class-based application handling full-time, part-time, and commission-based employee types using inheritance.

🧩 Class Hierarchy

  • Employee (Abstract) – Base class
  • FullTimeEmployee – BasePay + Allowance
  • PartTimeEmployee – DailyRate × Days
  • SalesEmployee – Inherits FullTime + Commission

💻 Application Features

  • Object creation and polymorphism
  • Interface-based pay calculation
  • Console-based CRUD operations
