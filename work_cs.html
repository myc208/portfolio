C# Employee Management Application

A desktop app that manages different employee types using class inheritance and pay calculation logic.

Objective

To demonstrate class hierarchy in C# through a system that manages Full-Time, Part-Time, and Sales employees, with pay calculations and basic CRUD functionality.

Technologies Used

  • C#
  • Visual Studio Code
  • .NET Framework

Structure

  • Employee (Abstract): Shared properties (ID, Name), `CalculatePay()`
  • FullTimeEmployee: Adds Basic Pay and Allowance
  • PartTimeEmployee: Uses Daily Rate x Days Worked
  • SalesEmployee: Inherits from FullTime, adds commission logic

Example – FullTimeEmployee.cs


class FullTimeEmployee : Employee {
    public double BasicPay { get; set; }
    public double Allowance { get; set; }

    public override double CalculatePay() {
        return BasicPay + Allowance;
    }

    public override string ToString() {
        return base.ToString() + $"\tPay: {CalculatePay()}";
    }
}
