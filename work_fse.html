Microsoft Excel Spreadsheet Engineering

Exploring excel spreadsheet formulas and functions such as macros, visualizations, vlookup, etc.

Project Description

Explore and familiarize with various formulas and functions within the spreadsheet that makes data analysis more efficient by reducing manual work for filtering, searching, and calculating. Learn visualization and storyboard creation techniques within the application to help in broadening data analysis choices of platforms.

Technologies Used

  • Google Spreadsheet
  • Python
  • Microsoft Excel
  • Provided sample dataset

Developments

  • Apply various formulas and functions within a spreadsheet
  • Extract values from another spreadsheet within the same file
  • Extract values from anther file
  • Create graphical representation of data
  • Customise data field for more in depth graphical analysis
  • Usage of macros

Example of formulas

  • "=SUM(A1:A10)": Adds all values from cell A1 to A10.
  • "=AVERAGE(B1:B5)": Finds the average of values in cells B1 through B5.
  • "=IF(C1 > 100, "Over 100", "Under 100")": If the value in cell C1 is greater than 100, it will return "Over 100"; otherwise, it returns "Under 100".
  • "=VLOOKUP(D1, A2:B10, 2, FALSE)": Looks for the value in D1 within the range A2:B10 and returns the value from the second column of the table.
  • "=COUNTIF(E1:E10, "Apple")": Counts how many times the word "Apple" appears in the range E1 to E10.
  • "=CONCATENATE(F1, " ", G1)": Joins the text in F1 and G1 with a space in between.
