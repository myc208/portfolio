Development of Recurrent Neural Network

Text classification and generation

Project Description

Develop a neural network model to learn from several text paragraphs and continue forming complete paragraphs of up to a certain number of words from an incomplete sentence or word input.

Technologies Used

  • Google Collab
  • Kaggle
  • Python
  • Provided dataset

Summary of Steps

  • Develop a base model
  • Analyse results
  • Tweaking of hyperparameters to improve base model based on results
  • Compare results with previous model(s)
  • Continue making changes until accuracy curve plateaus
  • Log and record each changes done and effect of the change
  • Use untrained data for testing
  • Record analysis results and findings
