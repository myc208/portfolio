Development of Convolutional Neural Network

Comparing self-trained models to pre-trained model (VGG16)

Project Description

Develop a neural network model for prediction accuracy comparison with other pre-trained learning algorithms such as VGG16, ResNet, InceptionV3, and MobileNet. The outcome of the project is to identify an algorithm that most accurately classifies an image to the name of the food that is shown within.

Technologies Used

Python
TensorFlow/Keras
Google Colab
Image Processing

Model Comparison

Self-trained CNN

  • Custom architecture
  • Smaller parameter count
  • Faster training time
  • Lower initial accuracy

VGG16

  • Pre-trained on ImageNet
  • Deep architecture (16 layers)
  • Transfer learning benefits
  • Higher accuracy

Key Findings

Performance Advantage

The VGG16 pre-trained model achieved 15% higher accuracy than the self-trained model, demonstrating the power of transfer learning

Training Efficiency

Despite the larger architecture, VGG16 converged faster due to pre-trained weights
