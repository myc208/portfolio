Development of Convolutional Neural Network

Designing a CNN from scratch and benchmarking it against VGG16 for food image classification.

Purpose

The aim was to evaluate model performance and speed across a custom CNN versus a pre-trained VGG16 model using the same dataset.

Environment

  • Python
  • Google Colab
  • TensorFlow
  • Kaggle Datasets

Steps Taken

  1. Build a simple CNN from scratch
  2. Compare structure and results to VGG16
  3. Log accuracy, loss, and inference time

Visual Comparison

Self-trained CNN model

VGG16 Pre-trained model

Result Comparison Chart

Analysis

  • Accuracy: VGG16 outperformed custom CNN on all metrics.
  • Training Time: VGG16 converged faster despite larger size due to pre-trained weights.
  • Architecture Depth: Custom CNN lacked deep enough convolutional layers for complex features.
