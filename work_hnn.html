Development of Hybrid Neural Network

Optimizing algorithm learning through convergence of convolutional layers with LSTM/GRU layers.

Overview

Develop a neural network model for prediction accuracy comparison with generative learning algorithms. The goal is to identify the best-performing method for phishing email detection (spam vs ham).

Technologies Used

  • Google Colab
  • Python
  • Kaggle Datasets

Workflow

  1. Identify relevant datasets from Kaggle
  2. Clean and merge data
  3. Build baseline model with CNN
  4. Incorporate RNN layers (LSTM/GRU)
  5. Evaluate performance vs other models
